## Bank Transaction Database: A schema for managing customer accounts and transactions

This Git repository stores the design and code for a bank transaction database. This schema allows you to track and manage customer bank accounts, including deposits, withdrawals, transfers, and service payments. 

Built with SQL, this database aims to:

*  Offer a central repository for efficient management and reporting of all customer transactions. 
*  Provide a foundation for building functionalities like account creation, transaction history viewing, and generating reports.

This project is a great opportunity to:

*  Solidify your understanding of relational database design principles.
*  Explore the power of SQL for managing financial data.
*  Build a practical and useful application.
